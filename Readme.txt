<<<HOTEL BROWSER>>>
Mi aplicación consiste en un buscador de hoteles, esta diseñada para los clientes unicamente sin funciones
de administración. En ella debemos iniciar sesión o registrarnos para posteriormente tener acceso a nuestro
perfil para cerrar sesión. Otra pestaña muestra una lista con cards de todos los servicios junto a la tercera
pestaña para manejar nuestras reservas. A su vez dispondremos de un botón para añadir una reserva con el que
desplegaremos un formulario en el que podremos seleccionar entre los servicios disponibles y rellenar más 
información. Nuestra navegación funcionará con un bottom navigation.
